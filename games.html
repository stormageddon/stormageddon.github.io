Games I've Made!

HORDE is a first person shooter that I made using Unity. The basis of the game was built as part of an excellent Udemy class on 3D game creation that you can find here. It is a very simple (and very rough) first person shooter, but it was mostly an attempt to learn basic game development and in that way I would call HORDE a wild success! Feel free to download a copy for free or play the WebGL version (coming soon).

There's bound to be bugs that I may or may not ever fix, but if you find issues, have suggestions, or just want to chat, feel free to reach out to me via email (`Get In Touch` button above), a GitHub issue, or tweet me @bad_wolf_inc.
